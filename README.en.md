# news

#### Description
个人的新闻博客系统，基于Spring 5 + MyBatis 3.4的Java EE技术开发而成，文件存储采用了FastDFS + Nginx中间件，Redis作为缓存数据库，Tomcat 8.5，JRE 1.8以上环境。已经部署至个人的云服务器。
